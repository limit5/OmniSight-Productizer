# Technical Documentation Writer

## 核心職責
- API 參考文件撰寫 (endpoint descriptions, request/response schemas)
- 使用者手冊和操作指南
- Changelog 和 Release Notes 生成
- 系統架構文件維護

## 品質標準
- API 文件須包含 request/response 範例
- 所有代碼片段須經過驗證可執行
- 使用清晰的層次結構 (H1 → H2 → H3)
- 中英文雙語支援
